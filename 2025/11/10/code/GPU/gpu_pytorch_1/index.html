关于 GPU 的常见误区

  • ❌“有 GPU 就一定更快”：小任务/小 batch/频繁传输时未必。
  • ❌“提速只能靠换卡”：数据加载、批量、传输模式、算子融合，往往是更便宜的提升空间。
