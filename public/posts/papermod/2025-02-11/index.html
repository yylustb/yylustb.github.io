Hugo搭建个人网页

February 11, 2025 · Aditya Telange

目录

  • 2.Hugo生成静态站点
  • 3.Hugo配置Theme
  • 4.Hugo本地部署
  • 5.Hugo部署到Github Pages

1.安装必要软件#

按照以下顺序安装软件（因为相互依赖关系）

  • Git

  • Powershell 7.x

    win11自带Powershell是5.1，需要更新到7.x。 在win11自带Powershell里输入以下指令，安装最新稳定版

    winget install --id Microsoft.Powershell --source winget

  • VS Code

  • Go

    go安装说明见官网 https://go.dev/doc/install

  • Github账户

  • choco

    参考官网 https://docs.chocolatey.org/en-us/choco/setup/ 安装choco，执行以下指令

    Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))

  • Hugo

    执行以下指令，利用 choco 安装 hugo

    choco install hugo-extended

  • Dart Sass

    执行以下指令，利用 choco 安装 Sass

    choco install sass

2.Hugo生成静态站点#

Powershell 切换至文件夹 A，执行以下指令

cd A
hugo new site demo

执行完上面指令后，hugo 会生成网站，保存路径为 A/demo。

注意：打开文件夹 A/demo，会发现hugo生成了很多文件，其中最重要的是配置文件 hugo.toml。 网站的配置文件，会略有区别。常见的配置文件有

  • hugo.yaml / hugo.toml
  • config.yaml / config.toml

切换路径至 A/demo，执行 git 初始化，为后续步骤做准备。

git init

3.Hugo配置Theme#

安装 Hugo 主题，参考官网 Hugo themes，接下来我们以 ananke 为例

安装 Hugo 主题 Ananke，执行以下指令

git submodule add https://github.com/theNewDynamic/gohugo-theme-ananke.git themes/ananke

主题 ananke 会存放在路径 A/demo/themes/ananke。 配置主题，执行以下指令

echo "theme = 'ananke'" >> hugo.toml

执行完上面指令后，网站配置文件最后一行会添加以下内容

1

theme: ["PaperMod"]

4. 添加网页内容#

在路径 A/demo 下，执行以下指令

hugo new content posts/hello.md

执行完上面指令后，hugo 会生成网页，保存路径为 A/demo/content/posts/hello.md。打开该文档，会看到以下内容

1
2
3
4
5

+++
date = '2025-02-11T07:14:09+08:00'
draft = false
title = 'My First Post'
+++

5.Hugo本地部署#

如果 draft = true，执行以下指令（以下两个指令是一样的，只执行一个即可）

hugo server -D 
hugo server --buildDrafts

如果 draft = false，

hugo server

最后，可以在浏览器输入地址 http://localhost:1313/，即看到网页内容
